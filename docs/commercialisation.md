# Commercialisation

The free MVP proves core operational value: map intelligence, forecasts, vessel recommendations, and collection workflows.

Paid paths:

- Resort subscriptions for site risk monitoring, alerts, and reports
- Coastal authority contracts for regional monitoring and response planning
- Marine contractor tools for vessel dispatch and job costing
- Premium vessel operations module with routing, telemetry, and task history
- White-label dashboards for destination management groups
- API data licensing for insurers, ports, and environmental monitoring groups
- Future robotic collection licensing for autonomous collection fleets

Payments are intentionally not implemented in the MVP. The data model prepares organisations, users, plans, API access, and role separation for a later billing layer.
