# API

FastAPI serves OpenAPI docs at `/docs`.

Core endpoints:

- `GET /api/health`
- `GET/POST /api/observations`
- `GET /api/patches`
- `POST /api/patches/from-observations`
- `GET /api/predictions`
- `POST /api/predictions/run`
- `GET /api/risk-zones`
- `GET /api/client-sites`
- `GET /api/vessels`
- `GET/POST /api/vessel-positions`
- `GET /api/routes`
- `GET /api/routes/recommendations`
- `GET/POST /api/tasks`
- `GET /api/alerts`
- `POST /api/uploads/geojson`
- `POST /api/demo/seed`
- `GET /api/autonomous/tasks`
- `POST /api/autonomous/status`
- `POST /api/autonomous/telemetry`
